Demo

Written in Native Angular 10-13/Vanilla JavaScript, with minimal libraries

You can use the contacts ribbon button to open the demo contact manager, this is just a work in progress demo which has some functions that work, test at own peril. The contact manager utilises my own PHP API which is also used for login/session control, and authentication token issue.

The frame work uses full lazy loading of components so please be sure to watch the Network Activity monitor in the console as you navigate some of the demo items. All icons an assets are created by me, except for the demo video which is courtesy of Mesh More Info

The back end API is written in PHP and utilises JWT's

Please scroll down this window to see the demo window and panel options/controls/methods.
